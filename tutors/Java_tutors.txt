javac - Консольная команда для компиляции/сборки файла с расширением java.
java - Консольная команда для запуска файла с расширением class.

Типы данных в Java:
	Целые числа:
		byte - 1 байт
		short - 2 байт
		int - 4 байт
		long - 8 байт
	Вещественные числа:
		float - 4 байт (одинарной точности)
		double - 8 байт (двойной точности)
	Символ:
	 	char - 2 байт
	Логическое значение:
		boolean - 1 байт
